# 📊 Data Professional Survey Analysis Dashboard

---

## 📌 Project Overview

This Power BI project analyzes survey responses from 630 data professionals worldwide to understand industry trends, career satisfaction, salary distribution, and programming preferences.

The dashboard transforms raw survey data into actionable insights about:

- Industry entry difficulty  
- Salary distribution by role  
- Work-life balance satisfaction  
- Salary happiness levels  
- Preferred programming languages  
- Geographic distribution of respondents  

---

## 🌍 Dataset Summary

- Total Survey Participants: **630**
- Average Age: **29.87 years**
- Countries Represented: United States, India, United Kingdom, Canada, and Others
- Roles Included:
  - Data Scientist
  - Data Engineer
  - Data Architect
  - Data Analyst
  - Database Developer
  - Students / Entry-Level

---

## 🎯 Business Problem

The data industry is growing rapidly, but questions remain:

- How difficult is it to break into data careers?
- Which roles offer the highest salaries?
- Are professionals satisfied with their compensation?
- How balanced is their work-life experience?
- Which programming languages dominate the industry?

This dashboard answers these questions using interactive data visualization and KPI modeling.

---

## 🛠️ Tools & Technologies Used

## 📊 Key Metrics & Visualizations

### 1️⃣ Industry Entry Difficulty

Breakdown of how professionals rated the difficulty of entering the data field:

- 42.7%: Neither Easy nor Difficult
- 24.7%: Difficult
- 21.2%: Easy
- 6.9%: Very Difficult
- 4.3%: Very Easy

Insight: Most respondents consider entering the industry moderately challenging.

---

### 2️⃣ Geographic Distribution

The majority of respondents are from:

- United States (largest share)
- India (second largest group)
- Followed by UK, Canada, and others

This highlights strong participation from major tech-driven economies.

---

### 3️⃣ Preferred Programming Language

Top languages among data professionals:

- 🥇 Python (dominant preference)
- R
- Other
- C/C++
- JavaScript
- Java

Python clearly leads as the industry standard for data roles.

---

### 4️⃣ Average Salary by Job Title

Highest average salaries:

1. Data Scientist  
2. Data Engineer  
3. Data Architect  
4. Other Technical Roles  
5. Data Analyst  
6. Database Developer  
7. Students / Entry-Level  

Insight: Advanced technical roles command significantly higher compensation.

---

### 5️⃣ Work-Life Balance Happiness

Average Rating: **5.74 / 10**

Indicates moderate satisfaction with work-life balance across the industry.

---

### 6️⃣ Salary Happiness

Average Rating: **4.27 / 10**

Insight: Professionals are less satisfied with salary compared to work-life balance.

---

## 📈 Key Insights

- Breaking into data careers is moderately challenging but not extreme.
- Python is overwhelmingly the preferred language.
- Data Scientists earn the highest average salaries.
- Salary satisfaction is relatively low compared to work-life balance.
- The industry appears competitive yet financially rewarding for senior roles.

## 🔄 Project Workflow

### Phase 1 – Data Preparation
- Cleaned survey dataset in Power Query.
- Handled null values and standardized role names.
- Modeled relationships between survey fields.

### Phase 2 – DAX & KPI Development
- Created measures for:
  - Average Age
  - Average Salary
  - Happiness Scores
  - Respondent Count
- Designed ranking visuals for salary comparison.

### Phase 3 – Visualization & Design
- KPI Cards for high-level metrics.
- Donut Charts for categorical breakdown.
- Bar Charts for salary and language comparison.
- Tree Map for country distribution.
- Gauge Charts for satisfaction indicators.
